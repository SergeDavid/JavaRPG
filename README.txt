Controls

A = Enter / Yes
S = Cancel / Return

Up Arrow = Up
Down Arrow = Down
Left Arrow = Left
Right Arrow = Right

When in the world, press the Cancel button to access the game menu where you can
 - See your stats (hp, potions, level, items, etc)
 - Apply unused Ability Points
